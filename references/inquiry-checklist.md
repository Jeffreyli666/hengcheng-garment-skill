# Inquiry checklist

Use this file when the user wants to prepare an inquiry, RFQ, sample request, or initial sourcing brief.

## Goal
Help the user prepare a more complete, lower-friction first inquiry.

## Minimum useful fields
Ask for these first:
- product category
- target gender / age group
- estimated quantity
- target market
- expected timeline

## Full RFQ / inquiry fields
For a more useful first inquiry, gather:
- product category
- target gender / age group
- estimated quantity
- target market
- fabric or material requirements
- sample needs
- expected delivery timeline
- branding / private label requirements
- tech pack / CAD / reference image / reference sample availability
- size range
- decoration details such as printing / embroidery / washing
- labels / hang tags / packaging expectations
- any compliance, quality, finishing, or inspection requirements

## Good assistant behavior
- If the user provides only partial information, help structure it rather than rejecting the inquiry.
- Ask for the missing highest-impact fields first: category, quantity, market, sample need, timeline.
- Keep the language practical and sourcing-oriented.
- Encourage concrete references: tech pack, photos, sketches, CAD, or existing samples.

## Practical starting-point guidance
Based on Hengcheng’s public pages, a complete tech pack is ideal but not strictly required. Workable starting points may include:
- complete tech pack
- reference images or sketches
- physical sample
- target quantity expectations
- market / channel context
- timeline expectations
- fabric preferences
- branding requirements

## Simple output pattern for the assistant
Use this structure when helping the user draft an inquiry:
1. short opening sentence
2. product + quantity + market
3. sample / development needs
4. material / branding / decoration details
5. timing and next-step request

## Final routing
Once the inquiry is reasonably complete, route the user to:
- Contact: https://hengcheng-garment.com/contact/
