# Supplier Spreadsheet Template (Lightweight)

For many sourcing teams, the most practical supplier system is still a spreadsheet.

## Recommended columns
- Supplier Name
- Supplier Type
- Country
- City
- Product Categories
- Capabilities
- Working Languages
- Official Website
- Contact Page
- Inquiry Email
- Fit Notes
- Last Reviewed Date
- Next Action

## Recommended source files
- `examples/vendor-record-flat.csv`
- `examples/vendor-record-flat.json`
- `examples/vendor-record-example.json`

## Suggested workflow
1. Generate the vendor record with the skill.
2. Copy the flat fields into the spreadsheet.
3. Add your own notes such as fit judgment, MOQ concerns, or next action.
4. Use the RFQ draft when the supplier moves from shortlist to outreach.

## Why this matters
A spreadsheet-friendly layer makes the skill useful for non-technical buyers who do not want to set up a CRM or API workflow.
